# Vimium C config

This is my personal configuration for [Vimium C](https://github.com/gdh1995/vimium-c). (a fork of Vimium).
This config is not compatible with the original Vimium.

## Installation

1. Get the `config.json` file in this directory.
2. Open the Vimium C options page.
3. Click on the "Import settings" button.
4. Select the `config.json` file you downloaded.

## Keybinds

Keys are case sensitive. Capital letters require <kbd>⇧ Shift</kbd> to be pressed.

Multiple keys require to be pressed rapidly one after the other.

### Link hints

- <kbd>f</kbd> - Open link in current tab
- <kbd>F</kbd> - Open link in new tab and switch to it
- <kbd>yf</kbd> - Open link in new tab silently

### Scrolling

- <kbd>w</kbd>, <kbd>a</kbd>, <kbd>s</kbd>, <kbd>d</kbd> - Scroll up, left, down, right
- <kbd>W</kbd>, <kbd>S</kbd> - Scroll half a page up, down
- <kbd>x</kbd> - Scroll to the top of the page
- <kbd>X</kbd> - Scroll to the bottom of the page

### Tabs & History

- <kbd>q</kbd>, <kbd>e</kbd> - Switch to previous, next tab
- <kbd>Q</kbd>, <kbd>E</kbd> - Move current tab to the left, right
- <kbd>yq</kbd> - Close current tab
- <kbd>ye</kbd> - Create new tab
- <kbd>YQ</kbd>, <kbd>YE</kbd> - Close all tabs on the left, right

- <kbd>t</kbd>, <kbd>gh</kbd> - Go back, forward in history

### Misc

- <kbd>r</kbd> - Reload current page
- <kbd>i</kbd> - Toggle insert mode
- <kbd>I</kbd> - Toggle view sources

- <kbd>c</kbd> - Copy current URL
- <kbd>C</kbd> - Copy URL to a link (using Link Hints system)

- <kbd>p</kbd> - Zoom in
- <kbd>o</kbd> - Zoom out
- <kbd>P</kbd> - Reset zoom

### Vomnibar

- <kbd>v</kbd> - Open Vomnibar (open URL, bookmark, history, etc.)
- <kbd>V</kbd> - Open Vomnibar in new tab
- <kbd>yv</kb> - Edit current URL in Vomnibar
